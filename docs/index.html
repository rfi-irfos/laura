LAURA digitaler selbstschutz
worum es geht
ein köder-ordner, den ein täter findet statt deiner echten fotos.
proximity-tools (NFC/Bluetooth-basierte geräte, missbrauchte datenübertragung zwischen handys, physischer kurzzugriff) werden zunehmend genutzt, um unbemerkt private fotos von handys zu ziehen — oft gefolgt von erpressung. LAURA erzeugt dir einen ordner, der wie dein privater fotobereich aussieht. greift jemand ohne deine erlaubnis darauf zu, bekommt er statt echter fotos einen köder — und du bekommst einen beleg, dass es passiert ist.
was das tool tut und was nicht:
  • ✓ erzeugt einen download mit köder-dateien, die du selbst auf dein handy legst
  • ✓ zeichnet auf, wenn der köder später irgendwo geöffnet wird (zeit, technische kennung)
  • ✓ nur du kannst mit deinem code nachsehen, ob/wann ausgelöst wurde
  • ✗ greift nicht aktiv auf das gerät zu, das den köder öffnet — kein exploit, kein code, der dort läuft
  • ✗ meldet nichts automatisch an polizei oder dritte — das entscheidest du, ein mensch sieht sich jeden treffer erst an
  • ✗ braucht keine registrierung, keinen klarnamen, keine app-installation
ordner-name (der köder soll wie dein privater bereich aussehen)
dein code (merken oder screenshot — wird nirgends gespeichert, nur hier gezeigt)
nachschauen ob ausgelöst
ordner im zip
  1. zip entpacken (auf dem handy: die eine .html-datei direkt in Dateien öffnen — kein entpacken nötig)
  2. den entpackten ordner / die .html in deine fotos/downloads verschieben
  3. fertig — nichts weiter zu installieren
